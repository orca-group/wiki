## Building

**Prerequisites**

* [`Rust`](https://www.rust-lang.org/)
* [`mdBook`](https://crates.io/crates/mdbook)  

```sh
# Optionally, clean directories before building
$ mdbook clean

# For production
$ mdbook build

# For local development
$ mdbook serve
```
